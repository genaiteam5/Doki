# DOKI — 물건 도깨비

쓸모를 잃은 물건에 도깨비가 깃든다는 컨셉의 인터랙티브 브랜드 사이트.
`foundfounded.com` 무드의 미니멀 레이아웃 — 넓은 여백과 볼드한 타이포.

## 인터랙션

- **눈 그래픽 6개** 가 홈 화면에 감각적으로 흩어져 배치됩니다.
- 모든 눈동자는 **마우스 커서를 스르륵 따라옵니다** (eye-tracking, 부드러운 lerp).
- 6개 중 **3개는 진짜 '도깨비 문'**, **3개는 가짜**입니다.
  - **진짜 눈** 클릭 → 눈동자가 거대해지며 그 안으로 빨려 들어가는 **Zoom-in 전환** 후
    선풍기 · 우산 · 책 3가지 제품의 **그리드 갤러리**가 등장.
  - **가짜 눈** 클릭 → 깜빡임 · 눈동자 굴리기 · 사시 · 회전 등 **장난스러운 가짜 피드백**
    과 도깨비의 약 올리는 한 마디(토스트). 도깨비한테 속은 느낌!
- `Esc` / `← 눈으로 돌아가기` 로 홈 복귀.

## 기술

- 순수 **HTML / CSS / JavaScript**
- 애니메이션 **GSAP 3** (CDN)
- 눈·제품 그래픽은 인라인 **SVG** (`assets/`)
- 빌드 도구 없음 — 정적 파일 그대로 동작

## 로컬 실행

```bash
node server.js          # http://localhost:5500
# 또는 임의의 정적 서버로 index.html 서빙
```

## 구조

```
index.html        홈 + 갤러리 마크업
css/style.css     디자인 토큰 · 레이아웃 · 눈/카드 스타일
js/main.js        눈 생성 · 커서 추적 · 가짜 트릭 · 줌 전환
assets/*.svg      제품 일러스트 (선풍기 / 우산 / 책)
server.js         의존성 없는 로컬 미리보기 서버
```
